# Deutsche Arbeitsentwuerfe

Dieser Ordner enthaelt deutschsprachige Arbeitsfassungen und historische
Generierungsskripte. Diese Dateien sind nicht Teil der englischen
Zenodo-Release-Kandidatenpfade.

Die release-relevanten Dateien bleiben in `paper/`, `figures/`, `data/`,
`models/`, `scripts/`, `README.md`, `CITATION.cff` und `.zenodo.json` auf
Englisch.
