# Epistemic Status Model

This model follows the current paper draft and separates the theoretical
CPT-isomorphic structure from the experimental status of each antimatter system.

## Status Vector

Each antimatter entry is represented as:

```text
S(overline(E_Z)) = (S_name, S_nuc, S_atom, S_spec, S_chem, S_astro)
```

- `S_name`: terminology and notation status.
- `S_nuc`: antinucleus status.
- `S_atom`: neutral anti-atom realization status.
- `S_spec`: spectroscopy or precision-characterization status.
- `S_chem`: anti-ion, anti-molecule, or reaction-chemistry status.
- `S_astro`: astrophysical antimatter-search status.

## Levels

- `0`: not defined or not evidenced.
- `1`: theoretically expected from CPT/QED and matter chemistry.
- `2`: experimentally observed or produced.
- `3`: precisely characterized by quantitative measurements.

## Minimal Current Classification

- `overline(H)`: `(3, 3, 3, 3, 1-2, 0)`.
- `overline(He)`: `(2, 2, 0, 0, 0-1, 0)`.
- `overline(Li)` to `overline(Ne)`: `(1, 0-1, 0, 0, 0-1, 0)`.
- `overline(Na)` to `overline(Og)`: `(1, 0, 0, 0, 0-1, 0)`.

The table intentionally distinguishes observed antinuclei from neutral
anti-atoms.
